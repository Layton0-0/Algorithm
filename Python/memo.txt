파이썬 메모

파이썬은 다른 언어들과 다르게 스트립트가 실행됐을 때 메인 함수가 실행되지 않는다. 없기도 하고.
들여쓰기가 되지 않은 level0의 코드를 모두 실행하는데 함수나 클래스는 이때 정의되지만 실행은 되지 않는다.

파이썬에서는 대개 스네이크표기법을 사용한다.

내장함수
__init__
  -> 생성자 역할
  -> 클래스를 호출하였을 때 가장 먼저 실행되는 함수
self
  -> 클래스의 지역변수

내장변수
__name__
  -> 직접 실행된 모듈의 경우 값 = __main__
  -> 직접 실행되지 않은 import된 모듈은 모듈의 이름(파일명)을 가지게 됨. (travel.thailand) not class name
